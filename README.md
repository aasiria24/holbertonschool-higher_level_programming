## ğŸ“‹ Task List

1. **Print String**: Write a Python script that prints exactly "Programming is like building a multilingual puzzle"
2. **Print Integer**: Print the integer stored in a variable followed by "Battery street"
3. **Print Float**: Print a float with precision of 2 digits
4. **String Repetition**: Print a string 3 times followed by its first 9 characters
5. **String Concatenation**: Concatenate strings to print "Welcome to Holberton School!"
6. **String Slicing**: Extract and print first 3 letters, last 2 letters, and middle of a string
7. **Advanced Slicing**: Extract and concatenate substrings without creating new variables
8. **The Zen of Python**: Print "The Zen of Python" using Python's Easter egg
